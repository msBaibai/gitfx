简介

Framework7用于构建iOS，Android和桌面应用程序的全功能框架

为各平台构建应用

借助Framework7提供的出色的UI组件集，它可以创建具有原生外观的Web应用程序
1、渐进式Web应用程序（PWA）
2、iOS和Android应用程序
3、Framework7与Electron和NW.js等额外工具结合使用，可以构建本机桌面应用程序

使用习惯用的框架和打包工具

1、jquery
2、vue.js
3、React
4、gulp
5、rollup.js
6、webpack

Framework7生态系统

1、Framework7 Core
2、Framework7 Vue
3、Framework7 React
4、Framework7 CLI
5、Framework7 Icons
6、Template7
7、dom7

开始构建项目

npm i framework7-cli cordova -g

framework7 create --ui
